# Images des Cartes Pokémon

## Spécifications

### Dimensions
- **Largeur:** 300px
- **Hauteur:** 363px
- **Format:** PNG avec transparence (recommandé)
- **Ratio:** 1:1.21

### Nommage des fichiers

Les cartes doivent être nommées avec leur ID :
- `card_1.png` à `card_50.png` (pour le premier booster)

### Fichier spécial

- `card_back.png` : **Dos de carte** utilisé pour les cartes non possédées dans la collection
  - Même dimensions : 300x363px
  - Design uniforme pour toutes les cartes non découvertes

## Répartition des raretés (Booster 1)

- **Cartes 1-30** : Commun (gris/blanc)
- **Cartes 31-42** : Peu commun (vert)
- **Cartes 43-47** : Rare (bleu)
- **Cartes 48-49** : Épique (violet)
- **Carte 50** : Légendaire (orange)

## Design recommandé

- Le numéro de la carte peut être intégré dans le design
- Ajouter la rareté visuellement (bordure, effet, etc.)
- Garder un style cohérent pour tout le set

## Remplacement

Pour remplacer les placeholders :
1. Préparez vos 50 cartes + le dos
2. Renommez-les selon la convention (card_X.png)
3. Remplacez les fichiers dans ce dossier
4. Redémarrez le bot

---

**Note :** Les placeholders actuels sont des images de test. Remplacez-les par vos vrais designs !
