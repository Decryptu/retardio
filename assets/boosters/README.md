# Images des Boosters

## Spécifications

### Dimensions
- **Largeur:** 300px (ou plus si design détaillé)
- **Hauteur:** 420px (même ratio que les cartes)
- **Format:** PNG avec transparence (recommandé)

### Nommage des fichiers

Les boosters doivent être nommés avec leur ID :
- `booster_1.png` (Édition Première)
- `booster_2.png` (futur booster)
- etc.

## Design recommandé

- Design attractif représentant le thème du booster
- Logo ou nom du set
- Visuel qui donne envie d'ouvrir le pack !

## Utilisation

Cette image est affichée :
- Lors de la sélection du booster à ouvrir
- Dans les messages de confirmation
- Potentiellement dans l'animation d'ouverture

## Remplacement

Pour remplacer les placeholders :
1. Préparez votre design de booster
2. Renommez-le `booster_1.png`
3. Remplacez le fichier dans ce dossier
4. Redémarrez le bot

---

**Note :** Le placeholder actuel est une image de test. Remplacez-le par votre vrai design !
